# CLI Password Manager

A simple command-line password manager built with Python.

## Description
This project allows users to store and retrieve passwords securely using encoding.

## Features
- Add passwords
- View stored passwords
- Encoded storage using Base64
- Simple CLI interface

## Technologies Used
- Python
- base64 module

## How to Run
1. Clone the repository
2. Navigate to the project folder
3. Run:

   python password_manager.py

4. Choose options from menu
